*Return Negative*

```javascript
function makeNegative(num) {
  if(num <= 0) {
    return num
  } else {
    return -num;
  }
}
```

*Sum of positive*
https://www.codewars.com/kata/5715eaedb436cf5606000381
```javascript
function positiveSum(arr) {
  let res = 0;
  for(let i = 0; i < arr.length; i++){
    if(arr[i] > 0) {
    res += arr[i];
    }
  }
  return res;
}
```
